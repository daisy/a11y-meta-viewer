Accessibility Metadata Viewer

How to use

The DAISY Accessibility Metadata Viewer converts the accessibility metadata in an EPUB package document or ONIX record into display statements in accordance with the W3C Accessibility Metadata Display Guide for Digital Publications 2.0.

There are currently two ways to view metadata:

  • Copy and paste an EPUB package document or ONIX metadata record into the text box.
  • Click on the link for preconfigured files and select one of the options. A sample file will automatically be loaded in the text box.

Please note that no information about your publication is collected or stored by this application. The data you load into the text box is processed locally in your browser by JavaScripts.

If you prefer to run the application code locally, you can check out the viewer source from the GitHub repository. You will need to configure the application to run in a local web server such as Apache or IIS in this case.
